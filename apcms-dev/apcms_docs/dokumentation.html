Neueste Version downloaden

 Back to Top

Die neueste Version dieser Applikation finden Sie immer auf unserer Homepage:
www.php-programs.de

Achten Sie explizit auf die Versionsnummer der verfügbaren Versionen und auch auf das Release-Datum, damit Sie immer die aktuellste Version downloaden.
Später, wenn Sie die Applikation einmal installiert haben, haben Sie dort im Admincenter die Möglichkeit, automatisch immer die aktullste Version downzuloaden und auch gleich vom System updaten zu lassen.



Archiv entpacken

 Back to Top

Wenn Sie sich die neueste Version heruntergeladen haben, haben Sie eine Datei, die ungefähr solch einen Dateinamen hat: "apcms-1.0.0a-20040516.zip".
Diese Datei können Sie mit jedem zip-tauglichen Entpacker (WinZIP, pkunzip, WinRAR, WinACE, etc.) entpacken. Lesen Sie dazu die Dokumentation Ihres Entpack-Programms.
Haben Sie das Archiv erfolgreich entpackt, so haben Sie nun ein neues Verzeichnis im gleichen Ordner, wo das Archiv drin liegt.
Dieses Verzeichnis hat i.d.R. denselben Namen wie das Archiv, allerdings ohne die Endung ".zip".
Darin wiederum finden Sie zwei weitere Verzeichnisse, die da lauten: "apcms" und "docs".



Hochladen auf den Webserver

 Back to Top

Um eine Applikation, bzw. deren Dateien und Verzeichnisse auf einen Webserver hochzuladen, benötigen Sie einen sogenannten FTP-Client, also ein Programm, das das Übertragungsprotokoll "FTP" beherrscht.
Beispiele für solche FTP-Clienten: TotalCommander, WS_FTP, FlashFTP, gFTP, ...
Wie Sie Ihren FTP-Client benutzen, um eine Applikation auf Ihren Webserver hochzuladen, entnehmen Sie bitte der, dem Programm beiliegenden Dokumentation.

Um eine PHP-Applikation ohne Probleme hochzuladen, müssen Sie Ihren FTP-Client so einstellen, das er selber die Methode (ASCII oder Binaer) des Uploads wählt, also die Methode "Auto" oder "Automatisch" wählen.

Beachten Sie, das es eine grosse Rolle spielt, wo Sie dieses Verzeichnis auf Ihrem Webserver speichern.
Wollen Sie das APCMS später im Browser so aufrufen: http://www.ihredomain.de/apcms, so muss das Verzeichnis "apcms" direkt in das Root-Verzeichnis Ihrer bisherigen Webseite hochgeladen werden.
Wollen Sie Ihr APCMS aber direkt mit dem Domain-Namen erreichen, also z.B. so: http://www.ihredomain.de, so dürfen Sie nicht das Verzeichnis "apcms" hochladen, sondern müssen den gesamten Inhalt des Verzeichnisses "apcms" mit der Maus markieren und diesen Inhalt dann in das Root-Verzeichnis Ihrer bisherigen Webseite hochladen.

Laden Sie nun entweder das komplette Verzeichnis "apcms", welches sich unterhalb des aus dem Zip-Archiv entstandenen Verzeichnisses befindet, inklusive Inhalt, also inklusive aller darin befindlichen Dateien und Unterverzeiochnisse (rekursiv), oder nur den kompletten Inhalt des Verzeichnisses "apcms" mit Ihrem FTP-Client hoch.

Der Vorgang des Hochladens wird nun eine Weile dauern, je nachdem, über was für eine Internet-Verbindung Sie verfügen.
